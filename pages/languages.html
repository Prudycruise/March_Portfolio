Languages

Overview cards for major programming languages with use-cases and examples.

Python

Best for automation, data science, and backend APIs where readability and velocity matter.

Use-cases: Machine learning, scripting, web services.

def greet(name: str) -> str:
    return f"Hello, {name}!"

JavaScript

The language of the browser and modern full-stack apps via Node.js.

Use-cases: UI interactions, APIs, real-time apps.

const greet = (name) => `Hello, ${name}!`;

C++

High-performance systems language used in engines, finance, and low-level tooling.

Use-cases: Game engines, embedded systems, performance-critical apps.

#include <iostream>
int main() { std::cout << "Hello"; }
